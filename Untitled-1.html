Introduction Complète à Java

1. Qu'est-ce que Java ?

Java est un langage de programmation orienté objet, créé en 1995 par Sun Microsystems, maintenant détenu par Oracle. Il est connu pour sa portabilité grâce à la JVM (Java Virtual Machine), qui permet aux programmes écrits en Java de fonctionner sur toutes les plateformes supportant la JVM.

2. Les Bases de Java

Un programme Java commence toujours par une classe contenant une méthode principale main. Voici un exemple simple :

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

3. Les Types de Données

Java prend en charge plusieurs types de données :

  • Primitifs : int, float, boolean, char, etc.
  • Référencés : String, ArrayList, HashMap, etc.

Exemple :

int age = 25;
String name = "Alice";
ArrayList list = new ArrayList<>();
list.add("Java");

4. Programmation Orientée Objet

Java est basé sur les principes de la POO :

  • Encapsulation : Restreindre l'accès aux données via des modificateurs comme private.
  • Héritage : Permettre à une classe de réutiliser les propriétés d'une autre classe.
  • Polymorphisme : Une méthode peut se comporter différemment selon les contextes.
  • Abstraction : Masquer les détails d'implémentation pour se concentrer sur l'essentiel.

5. Gestion des Exceptions

Les exceptions permettent de gérer les erreurs pendant l'exécution d'un programme. Structure typique :

try {
    // Code pouvant générer une exception
} catch (ExceptionType e) {
    // Code pour gérer l'exception
} finally {
    // Code qui s'exécute toujours
}

6. Framework des Collections

Java propose un riche ensemble de collections pour stocker et manipuler des données :

  • List : ArrayList, LinkedList
  • Set : HashSet, TreeSet
  • Map : HashMap, TreeMap

7. Les Interfaces

Une interface définit un contrat que les classes implémentent. Elles contiennent des méthodes abstraites, mais depuis Java 8, elles peuvent également inclure des méthodes default et static.

interface Animal {
    void manger();
    default void dormir() {
        System.out.println("Je dors.");
    }
}

class Chat implements Animal {
    public void manger() {
        System.out.println("Le chat mange.");
    }
}

8. Les Exceptions Personnalisées

Vous pouvez créer vos propres exceptions en étendant la classe Exception :

class MonException extends Exception {
    public MonException(String message) {
        super(message);
    }
}

public class TestException {
    public static void main(String[] args) {
        try {
            throw new MonException("Erreur personnalisée.");
        } catch (MonException e) {
            System.out.println(e.getMessage());
        }
    }
}
